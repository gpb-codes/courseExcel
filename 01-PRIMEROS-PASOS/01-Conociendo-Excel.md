# ¿Qué es Excel?

Microsoft Excel es una **hoja de cálculo** profesional que forma parte de Microsoft 365. En esencia, es una cuadrícula electrónica gigante donde puedes almacenar, organizar, calcular y analizar datos. Desde llevar un presupuesto familiar hasta gestionar la contabilidad de una empresa multinacional, Excel es la herramienta de productividad más utilizada en el mundo empresarial.

## La ventana de Excel: anatomía completa

Cuando abres Excel, te encuentras con varios elementos cuidadosamente diseñados. Conocerlos es el primer paso para dominar la herramienta.

### 1. Cinta de opciones (Ribbon)

Es la barra horizontal superior con pestañas. Cada pestaña agrupa comandos relacionados:

| Pestaña | Funciones principales | Cuándo usarla |
|---------|----------------------|---------------|
| **Inicio** | Portapapeles, fuente, alineación, números, estilos, celdas, edición | El 80% de tu trabajo diario |
| **Insertar** | Tablas dinámicas, gráficos, imágenes, formas, encabezados | Cuando necesitas elementos visuales |
| **Diseño de página** | Márgenes, orientación, área de impresión, temas | Antes de imprimir o exportar |
| **Fórmulas** | Insertar función, biblioteca de funciones, nombres definidos | Para crear cálculos |
| **Datos** | Ordenar, filtrar, Power Query, conexiones externas | Cuando trabajas con grandes volúmenes de datos |
| **Revisar** | Ortografía, comentarios, proteger hoja/libro | Para revisar y asegurar tu trabajo |
| **Vista** | Zoom, inmovilizar paneles, cambiar entre vistas | Para personalizar cómo ves la hoja |
| **Ayuda** | Tutoriales, búsqueda inteligente, contacto con soporte | Cuando necesitas asistencia |

> **💡 Pro Tip:** Puedes minimizar la cinta con `Ctrl + F1` para ganar espacio vertical. Vuelve a presionarlo para mostrarla de nuevo. También puedes hacer doble clic en cualquier pestaña para alternar la vista.

### 2. Barra de fórmulas

Ubicada debajo de la cinta, muestra el contenido real de la celda seleccionada. Aquí puedes:
- Ver fórmulas completas sin truncarse
- Editar directamente el contenido de la celda
- Hacer clic en el botón `fx` para insertar funciones guiadas

### 3. Barra de acceso rápido

La barra pequeña en la esquina superior izquierda (arriba de la cinta). Contiene Guardar, Deshacer y Rehacer. Puedes personalizarla haciendo clic en la flecha hacia abajo al final.

> **💡 Pro Tip:** Agrega "Vista preliminar de impresión" y "Vista rápida" a tu barra de acceso rápido. Son herramientas que usarás constantemente.

### 4. La cuadrícula

Es el área de trabajo principal. Está compuesta por:

- **Columnas**: identificadas con letras (A, B, C, ..., Z, AA, AB, ...)
- **Filas**: identificadas con números (1, 2, 3, ..., 1,048,576)
- **Celdas**: la intersección de una columna y una fila (ej: A1, B5, Z100)
- **Nombre de celda**: se muestra en el cuadro de nombres (izquierda de la barra de fórmulas)

Excel tiene **1,048,576 filas** y **16,384 columnas** (la última columna es XFD). ¡Suficiente espacio para casi cualquier proyecto!

### 5. Hojas del libro

En la parte inferior ves pestañas como "Hoja1", "Hoja2", "Hoja3". Un **libro** nuevo viene con 1 hoja (o 3 según la versión). Puedes:

- Hacer clic en las pestañas para cambiar de hoja
- Arrastrarlas para reordenarlas
- Hacer clic derecho para opciones avanzadas

### 6. Barra de estado

La franja inferior muestra información contextual:
- **Modo**: Listo, Escribir, Editar, Punto de interrupción
- **Resúmenes rápidos**: selecciona varias celdas numéricas y verás Suma, Promedio y Conteo automáticamente
- **Zoom**: control deslizante en la esquina inferior derecha

## Navegación básica: mueve como un experto

Dominar la navegación te hará 10 veces más rápido:

| Acción | Método | Atajo de teclado |
|--------|--------|------------------|
| Seleccionar celda | Haz clic en ella | Flechas direccionales |
| Ir a A1 | -- | `Ctrl + Inicio` |
| Ir a última celda con datos | -- | `Ctrl + Fin` |
| Ir a celda específica | -- | `F5` (o Ctrl + I) |
| Mover entre hojas | Clic en pestañas | `Ctrl + AvPág` / `Ctrl + RePág` |
| Zoom | Ctrl + rueda del mouse | `Ctrl + rueda` |
| Abrir libro nuevo | -- | `Ctrl + U` |
| Abrir libro existente | -- | `Ctrl + A` |
| Guardar | -- | `Ctrl + G` |

## Escenario del mundo real

**Caso práctico: Control de gastos personales**

Imagina que quieres llevar un registro mensual de tus gastos. Con Excel puedes:

1. Abrir un libro nuevo
2. En la columna A: listar conceptos (Renta, Comida, Transporte, etc.)
3. En la columna B: escribir los montos
4. Al final, usar `=SUMA(B2:B10)` para obtener el total

Este es el tipo de tarea que Excel resuelve en segundos y que a mano tomaría minutos.

## Errores comunes del principiante

| Error | Consecuencia | Solución |
|-------|-------------|----------|
| No distinguir entre libro y hoja | Confusión al buscar datos | Recuerda: el libro es el archivo, las hojas son páginas dentro |
| Hacer clic en una celda y escribir sin verificar | Sobrescribes datos existentes | Presiona `F2` para editar sin borrar |
| Ignorar la barra de fórmulas | No ves si una celda tiene fórmula o valor | Siempre revisa la barra de fórmulas |
| Perder cambios por no guardar | Trabajo perdido | Acostúmbrate a `Ctrl + G` cada pocos minutos |
| Usar el mouse para todo | Lentitud excesiva | Aprende un atajo nuevo cada día |

## Ejercicio práctico completo

Sigue estos pasos para familiarizarte con el entorno:

1. **Abre Excel** desde el menú Inicio o el acceso directo.
2. **Identifica cada elemento**: señala con el dedo (mentalmente) la cinta, la barra de fórmulas, el cuadro de nombres, las pestañas de hojas, la barra de estado.
3. **Navega por las pestañas**: haz clic en Inicio, Insertar, Fórmulas, Datos. Observa cómo cambian los botones.
4. **Prueba el zoom**: mantén presionado `Ctrl` y gira la rueda del mouse hacia adelante y atrás.
5. **Selecciona celdas**: haz clic en A1, luego B5, luego D10. Mira cómo cambia el cuadro de nombres.
6. **Mueve entre hojas**: haz clic en "Hoja2", luego vuelve a "Hoja1".
7. **Selecciona varias celdas**: haz clic en A1, mantén presionado y arrastra hasta C5. Verás el rango "A1:C5" resaltado.
8. **Revisa la barra de estado**: selecciona varias celdas vacías y observa la barra de estado en la parte inferior.
9. **Cierra Excel** sin guardar (Archivo > Cerrar, o `Ctrl + F4`).

## Puntos clave

- Excel es una hoja de cálculo organizada en **libros** → **hojas** → **filas/columnas** → **celdas**
- La **cinta de opciones** es tu panel de control principal
- Cada celda tiene una **dirección única** (columna + fila)
- La **barra de fórmulas** muestra el contenido real de la celda
- Usa la **barra de estado** para ver resúmenes rápidos sin fórmulas
- Los **atajos de teclado** multiplican tu productividad
- **Guarda frecuentemente** con `Ctrl + G`
