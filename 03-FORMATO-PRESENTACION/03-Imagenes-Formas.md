# Imágenes, formas y elementos visuales

Aunque Excel es ante todo una herramienta de datos y números, los elementos visuales como imágenes, formas, iconos y cuadros de texto son esenciales para crear reportes profesionales, dashboards y presentaciones ejecutivas. Una imagen o forma bien colocada puede comunicar en segundos lo que una tabla tarda minutos en explicar.

## Insertar imágenes

### Cómo insertar imágenes

Insertar > Ilustraciones > Imágenes > **Este dispositivo** (desde tu PC)

| Origen | Cuándo usarlo |
|--------|---------------|
| Este dispositivo | Logos corporativos, fotos de productos, firmas escaneadas |
| Imágenes de archivo | Ilustraciones prediseñadas (requiere conexión a internet) |

### Formatos de imagen compatibles

| Formato | Características |
|---------|-----------------|
| PNG | Fondo transparente, sin pérdida, ideal para logos |
| JPG/JPEG | Comprimido, sin transparencia, fotos |
| GIF | Animado (se reproduce dentro de Excel) |
| SVG | Vectorial, redimensionable sin pérdida, cambia de color |
| BMP | Sin compresión, archivos muy grandes |
| WEBP | Formato web moderno, buena compresión |

## Ajustar y editar imágenes

Al seleccionar una imagen, aparece la pestaña contextual **Formato de imagen** con potentes herramientas de edición:

### Herramientas de ajuste

| Herramienta | Descripción | Ejemplo de uso |
|-------------|-------------|----------------|
| **Quitar fondo** | Elimina el fondo automáticamente (marca áreas a conservar/eliminar) | Logos con fondo blanco |
| **Correcciones** | Brillo y contraste predefinidos | Fotos oscuras o sobreexpuestas |
| **Color** | Saturación, tono, colorear (efecto de filtro) | Logos en escala de grises |
| **Efectos artísticos** | Filtros: lápiz, acuarela, mosaico | Imágenes decorativas de fondo |
| **Comprimir imágenes** | Reduce el tamaño del archivo | Libros con muchas fotos que pesan mucho |
| **Restablecer imagen** | Vuelve al estado original | Cuando los ajustes no quedaron bien |

### Herramientas de recorte

1. Selecciona la imagen > Formato de imagen > **Recortar**
2. Arrastra los bordes negros para seleccionar el área que quieres conservar
3. Vuelve a hacer clic en Recortar o presiona Enter

**Opciones avanzadas de recorte:**
- **Recortar según forma**: Corta la imagen con forma de círculo, rectángulo redondeado, estrella, etc.
- **Relación de aspecto**: Recorta forzando proporciones 1:1, 16:9, 4:3
- **Rellenar** o **Ajustar**: Cómo se adapta la imagen al marco de recorte

> **Pro Tip:** Para logos corporativos, usa PNG con fondo transparente. En Formato de imagen > Tamaño, establece la altura exacta (ej: 1.5 cm) para que todos los logos tengan el mismo tamaño en el reporte.

### Posicionamiento de imágenes

Una vez insertada la imagen, puedes controlar cómo se posiciona respecto a las celdas:

| Opción de posición | Comportamiento |
|--------------------|----------------|
| **Mover y cambiar tamaño con celdas** | La imagen se mueve y escala al modificar filas/columnas |
| **Mover pero no cambiar tamaño** | Se mueve con las celdas pero mantiene su tamaño |
| **No mover ni cambiar tamaño** | La imagen permanece fija aunque se modifique la hoja |

**Cómo configurarlo:** Clic derecho en la imagen > Tamaño y propiedades > Propiedades

## Formas

### Cómo insertar formas

Insertar > Formas. Verás una galería con decenas de formas prediseñadas:

| Categoría | Ejemplos | Uso en Excel |
|-----------|----------|--------------|
| Líneas | Flechas, líneas rectas, curvas, conectores | Señalar datos importantes, diagramas de flujo |
| Rectángulos | Rectángulo, rectángulo redondeado | Títulos decorativos, tarjetas de KPI |
| Formas básicas | Círculo, triángulo, corazón, estrella | Indicadores visuales, iconografía |
| Flechas de bloque | Flechas gruesas en todas direcciones | Tendencia: arriba (crecimiento), abajo (decrecimiento) |
| Ecuaciones | Paréntesis, llaves, raíces | Documentación de fórmulas matemáticas |
| Diagramas de flujo | Decisión, proceso, dato, documento | Mapeo de procesos |
| Cintas y estrellas | Cinta, cinta ondulada, explosión | Llamadas de atención |

### Formato de formas

Al seleccionar una forma, aparece la pestaña **Formato de forma** (similar a Formato de imagen):

| Herramienta | Descripción |
|-------------|-------------|
| **Relleno de forma** | Color sólido, degradado, textura, imagen de fondo |
| **Contorno de forma** | Color, grosor, estilo de línea (sólida, guiones, puntos) |
| **Efectos de forma** | Sombra, reflexión, resplandor, bisel, 3D, rotación |
| **Relleno de texto** | Color de fondo del texto dentro de la forma |
| **Contorno de texto** | Color del borde de las letras |
| **Efectos de texto** | Sombra, reflexión, resplandor en el texto |

> **Pro Tip:** Para KPI visuales en dashboards, usa rectángulos redondeados con texto en negrita y color de énfasis. Ejemplo: rectángulo azul con texto blanco "Ventas: $50,000" → parece una tarjeta de dashboard profesional.

### Añadir texto a las formas

1. Clic derecho sobre la forma > **Modificar texto**
2. Escribe el texto directamente
3. Usa las herramientas de Inicio > Fuente para dar formato

**Alineación del texto dentro de la forma:**
- Clic derecho > Formato de forma > Opciones de texto > Cuadro de texto
- Ajusta márgenes internos, alineación vertical, dirección del texto

## Cuadros de texto

### Insertar y usar cuadros de texto

Insertar > Texto > Cuadro de texto (Horizontal o Vertical)

A diferencia de las celdas, los cuadros de texto: 
- Flotan sobre la cuadrícula (no están limitados a celdas)
- Pueden tener formato de párrafo (sangría, interlineado, viñetas)
- Son ideales para anotaciones, comentarios explicativos y títulos largos

### Usos profesionales de cuadros de texto

| Uso | Descripción |
|-----|-------------|
| **Títulos principales** | Títulos largos que no caben en Celdas combinadas |
| **Notas aclaratorias** | "Los datos de enero son preliminares" |
| **Fuentes de datos** | "Fuente: Departamento de Finanzas, 2024" |
| **Disclaimer legal** | Texto legal al pie de reportes financieros |
| **Instrucciones** | "Complete las celdas amarillas con datos reales" |
| **Firmas electrónicas** | "Aprobado por: _____ Fecha: _____" |

### Dar formato a cuadros de texto (formato avanzado)

- **Relleno y contorno** (igual que formas): clic derecho > Formato de forma
- **Efectos de texto**: sombra, reflejo, iluminado
- **Ajuste del texto**: Desde Formato de forma > Opciones de texto > Cuadro de texto
  - Márgenes internos (izquierdo, derecho, superior, inferior)
  - Alineación vertical (superior, medio, inferior)
  - Dirección del texto (horizontal, rotado 90°, rotado 270°)
  - Ajustar tamaño de forma al texto (la forma se expande automáticamente)

## Iconos

### Insertar iconos vectoriales

Insertar > Ilustraciones > Iconos → se abre una galería con miles de iconos vectoriales clasificados por tema.

**Ventajas de los iconos sobre las imágenes normales:**
- Son **vectores** (SVG): se redimensionan sin perder calidad
- **Cambian de color**: Formato gráfico > Relleno de gráfico
- **Transparencia** nativa
- **Tamaño de archivo** mínimo

### Categorías de iconos más útiles en Excel

| Categoría | Iconos útiles |
|-----------|---------------|
| Personas | Usuarios, equipo, cliente, empleado |
| Negocios | Dinero, crecimiento, gráfico, análisis, documento |
| Flechas | Arriba/abajo/derecha/izquierda, tendencias |
| Tecnología | Laptop, móvil, nube, datos, red |
| Comunicación | Correo, teléfono, chat, notificación |
| Señales | Check, error, advertencia, información |
| Objetos | Calendario, reloj, lupa, engranaje, estrella |

> **Pro Tip:** Los iconos en Excel se comportan como **Gráficos vectoriales** (SVG). Después de insertarlos, puedes convertirlos a formas (Formato gráfico > Convertir a forma) para desagruparlos y editar cada parte individualmente.

## SmartArt

Insertar > Ilustraciones > SmartArt. Crea diagramas profesionales predefinidos:

| Tipo de diagrama | Cuándo usarlo |
|------------------|---------------|
| Lista | Mostrar información secuencial o agrupada |
| Proceso | Flujo de trabajo, pasos, cronología |
| Ciclo | Procesos circulares, retroalimentación |
| Jerarquía | Organigramas, estructura de equipo |
| Matriz | Relaciones de 4 elementos |
| Pirámide | Jerarquías, niveles, cantidades |

**Cómo personalizar SmartArt:**
1. Al seleccionar SmartArt, aparecen dos pestañas: **Diseño** (cambiar color y estilo) y **Formato** (formato individual)
2. En Diseño de SmartArt, puedes cambiar los colores al tema actual
3. Agrega o quita formas desde el panel de texto

## Alinear, organizar y agrupar objetos

### Alinear objetos

Cuando tienes múltiples formas, imágenes o cuadros de texto, alinearlos manualmente es tedioso. Usa:

**Formato de imagen/forma > Organizar > Alinear:**

| Opción | Resultado |
|--------|-----------|
| Alinear izquierda | Todos los objetos se alinean al borde izquierdo del que está más a la izquierda |
| Alinear centro | Centrado horizontal respecto al conjunto |
| Alinear derecha | Alineados al borde derecho del que está más a la derecha |
| Alinear superior | Todos se alinean al borde superior del más alto |
| Alinear medio | Centrado vertical |
| Alinear inferior | Alineados al borde inferior del más bajo |
| Distribuir horizontalmente | Espacio igual entre objetos horizontalmente |
| Distribuir verticalmente | Espacio igual entre objetos verticalmente |

### Organizar capas

**Traer al frente / Enviar al fondo:** Controla qué objeto se ve encima de otro.

| Opción | Qué hace |
|--------|----------|
| Traer al frente | La forma se muestra sobre todas las demás |
| Enviar al fondo | La forma se muestra detrás de todas las demás |
| Traer adelante | Sube una posición en la pila |
| Enviar atrás | Baja una posición en la pila |

**Panel de selección:** Inicio > Edición > Buscar y seleccionar > Panel de selección. Muestra TODOS los objetos de la hoja en una lista. Puedes: 
- Ocultar/mostrar cada objeto (ojo)
- Reordenar capas arrastrando
- Seleccionar objetos difíciles de hacer clic
- Cambiar nombre a los objetos para identificarlos

### Agrupar objetos

1. Selecciona múltiples objetos (mantén `Ctrl` mientras haces clic en cada uno)
2. Clic derecho > Agrupar > Agrupar (o Formato > Organizar > Agrupar)

**Beneficios de agrupar:**
- Mueves todo el grupo como una sola entidad
- Aplicas formato a todos los elementos a la vez
- Copias y pegas el conjunto completo
- Redimensionas todo proporcionalmente

**Desagrupar:** Clic derecho > Agrupar > Desagrupar (cuando necesites editar elementos individuales)

> **Pro Tip:** Mantén `Alt` presionado mientras mueves o redimensionas una forma para que se "pegue" a la cuadrícula de celdas. Esto asegura que la forma quede perfectamente alineada con los bordes de las celdas.

## Errores comunes

### Error 1: Imágenes o formas que no se imprimen
Una imagen puede verse bien en pantalla pero no aparecer al imprimir.

**Solución:** Clic derecho en la imagen > Tamaño y propiedades > Propiedades > Marca "Imprimir objeto". También revisa Diseño de página > Imprimir > configuración.

### Error 2: Mover sin querer objetos
Al hacer clic en una celda cerca de una forma, a veces seleccionas la forma en lugar de la celda.

**Solución:** Asigna un nombre a la forma y bloquéala desde Formato de forma > Tamaño y propiedades > Propiedades > "No mover ni cambiar tamaño". O agrupa todas las formas en una capa de fondo.

### Error 3: Imágenes que aumentan demasiado el tamaño del archivo
Una imagen de cámara (5 MB) puede hacer que tu archivo Excel pase de 50 KB a 5 MB.

**Solución:** Comprime las imágenes desde Formato de imagen > Comprimir imágenes. Antes de insertar, reduce la resolución de las fotos a 150 DPI (suficiente para pantalla).

### Error 4: Formas solapadas que ocultan información
Al superponer varias formas sin control de capas, una puede tapar completamente a otra.

**Solución:** Usa el Panel de selección (Inicio > Buscar y seleccionar > Panel de selección) para ver y reordenar todas las capas.

### Error 5: No agrupar objetos que deben moverse juntos
Si tienes un título con un rectángulo de fondo y un icono, moverlos por separado desalinea el diseño.

**Solución:** Siempre que varios elementos formen una unidad visual (título + fondo + icono), agrúpalos.

### Error 6: SmartArt que no se actualiza con los datos
SmartArt es estático. Si cambias los datos de tu tabla, el SmartArt no se actualiza automáticamente.

**Solución:** Para elementos dinámicos, usa gráficos de Excel (Insertar > Gráficos) en lugar de SmartArt.

## Ejercicio paso a paso: Dashboard con KPI visuales

### Escenario de negocio
Eres analista de negocios y necesitas crear un dashboard con indicadores visuales para la reunión mensual de dirección. Debes combinar formas, iconos y cuadros de texto para que el reporte sea informativo y atractivo.

### Parte 1: Preparar los datos
1. Abre un libro nuevo
2. En **A1:B7** escribe los datos de indicadores:

| Indicador | Valor |
|-----------|-------|
| Ventas | $450,000 |
| Utilidad | $89,000 |
| Clientes Nuevos | 234 |
| Satisfacción | 92% |
| Empleados | 48 |
| Crecimiento | 15% |

### Parte 2: Crear tarjetas de KPI con formas
1. Insertar > Formas > Rectángulo redondeado
2. Dibuja un rectángulo de aproximadamente 4 columnas de ancho por 3 filas de alto
3. Formato de forma:
   - Relleno: azul claro (Énfasis 1, 40% más claro)
   - Contorno: azul oscuro (Énfasis 1)
   - Efecto: Sombra (exterior > sombra paralela)
4. Copia el rectángulo 5 veces (tendrás 6 tarjetas)
5. Organiza las 6 tarjetas en 2 filas de 3 usando Formato > Alinear > Distribuir horizontalmente

### Parte 3: Añadir contenido a cada tarjeta
1. Clic derecho en la primera tarjeta > Modificar texto
2. Escribe: `Ventas` en una línea, `$450,000` en la siguiente
3. Selecciona "Ventas" → fuente 10 pt, normal
4. Selecciona "$450,000" → fuente 18 pt, negrita
5. Repite para las otras 5 tarjetas con sus valores

### Parte 4: Añadir iconos a las tarjetas
1. Insertar > Iconos > Negocios > selecciona icono de "Dinero" para Ventas
2. Reduce el tamaño a 1.5 x 1.5 cm
3. Cambia color: Formato gráfico > Relleno de gráfico > Azul oscuro
4. Coloca el icono a la izquierda del texto en la tarjeta
5. Repite para cada tarjeta con el icono apropiado

### Parte 5: Agregar flechas de tendencia
1. Insertar > Formas > Flecha de bloque hacia arriba (verde) para indicadores positivos
2. Para Satisfacción (92%) y Crecimiento (15%), usa flecha arriba verde
3. Ajusta tamaño a 0.8 x 1.5 cm, colócala a la derecha del valor
4. Para indicadores que pudieran ser negativos, prepara una flecha roja hacia abajo

### Parte 6: Título y agrupación
1. Insertar > Cuadro de texto > escribe "Dashboard Mensual - Febrero 2024"
2. Fuente 24 pt, negrita, color azul oscuro
3. Selecciona TODAS las tarjetas (mantén `Ctrl`) > Clic derecho > Agrupar > Agrupar
4. Guarda como `03-dashboard-kpi.xlsx`

### Parte 7: Alineación final
1. Mueve el cuadro de texto del título alineado al centro del dashboard
2. Inserta una línea horizontal debajo del título (Insertar > Formas > Línea)
3. Selecciona herramienta Alinear para distribuir equitativamente

## Aspectos destacados (Key Takeaways)

- **Imágenes** y **formas** hacen que los reportes de datos sean más comunicativos y profesionales
- Usa **PNG con transparencia** para logos y **SVG** (iconos) para gráficos vectoriales redimensionables
- **Agrupa objetos** que formen una unidad visual para moverlos y copiarlos como un solo elemento
- Mantén `Alt` presionado para alinear formas a la cuadrícula de celdas
- Usa el **Panel de selección** para gestionar objetos superpuestos y capas
- **SmartArt** es ideal para diagramas estáticos (procesos, organigramas); **Gráficos** de Excel son para datos dinámicos
- **Comprime las imágenes** para mantener el archivo Excel ligero
- Las formas con **efectos de sombra y bisel** dan un aspecto moderno tipo dashboard
